# Chrislord Portfolio

A responsive portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## Live Demo

[View Live Site](https://chrislord-portfolio.vercel.app/)

## Repository

[GitHub Repository](https://github.com/ClTheGreatt/Chrislord-portfolio)

## Overview

This portfolio presents my projects, certifications, technical background, and contact information in a clean and interactive format. It focuses on clear structure, polished motion, and strong project presentation.

## Features

- Responsive portfolio layout
- Animated hero and section transitions
- Project showcase with preview modals
- Certificate gallery with modal previews
- Dark and light theme toggle
- Contact form powered by EmailJS
- Custom cursor and background effects
- Reusable React components

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- EmailJS
- Lucide React
- React Icons
